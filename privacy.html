Privacy Policy

YouTube Scroll Fullscreen is a Chrome extension that enhances YouTube’s fullscreen experience by enabling flexible layouts, allowing scrolling, and providing a configurable scroll-to-top button.

Information Collection

This extension does not collect, transmit, or sell any personal information. All functionality runs locally in your browser.

Use of Chrome Storage

The extension uses Chrome’s storage API solely to save user preferences — such as whether the scroll-to-top button is enabled, how far you must scroll before it appears, and the scroll-back animation duration. These settings are stored locally or synced with your Google account through Chrome’s built-in sync. No data is sent to external servers.

Host Permissions

The extension only requests access to https://www.youtube.com/* so it can apply layout tweaks and run its content scripts on YouTube pages. It does not access or modify other websites.

Third Parties

This extension does not include third-party analytics, tracking, or advertising. The only external link is the “made by vibephoenix” footer, which opens my YouTube channel when clicked.

Changes to This Policy

Any future updates to this privacy policy will be reflected on this page. We recommend checking here periodically for the latest information.
